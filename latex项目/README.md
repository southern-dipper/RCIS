## 使用指南

1. **解压项目**
   ```bash
   unzip Safe_Planning.zip
   ```

2. **检查环境**
   ```bash
   python check_env.py
   ```

3. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

4. **运行A*脚本**
   ```bash
   python A_star_rpis.py
   ```

5. **运行RRT脚本**
   ```bash
   python RRT_rpis.py
   python quick_test.py
   ```
